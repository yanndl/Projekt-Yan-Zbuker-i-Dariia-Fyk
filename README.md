# Projekt Gry: Subway Student
Platformówka typu Endless Runner (Unity & GitHub)

## Cel projektu
Stworzenie gry typu "Endless Runner" inspirowanej mechaniką Subway Surfers. Celem jest opracowanie grywalnego prototypu z dynamicznym otoczeniem i systemem punktacji.

## 🛠 Technologie
* Silnik: Unity 2022.3.x (LTS)
* Język: C#
* System wersji: 6000.4.1f1

## 📋 Podział obowiązków (Zadania)
* Yan: Tworzenie i przygotowanie zasobów (Assety), modele 3D, tekstury i animacje.
* Dariia:  Programowanie mechanik (Ruch, skok), logika gry, system generowania poziomów i zarządzanie projektem w Unity.

## ⚙️ Założenia techniczne (MVP)
1. Ruch: Postać poruszająca się po trzech liniach (lewo, środek, prawo).
2. Przeszkody: System generowania przeszkód na drodze gracza.
3. Zbieranie przedmiotów: System punktowy (monety/bonusy).
4. UI: Licznik punktów oraz ekran końcowy (Game Over).
